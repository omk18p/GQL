1. Generate Lexer & Parser

antlr4 -Dlanguage=Cpp grammar/GQL.g4 -visitor -o generated/


2. Compile the Project

g++ -O0 -std=c++17 -I/usr/local/include/antlr4-runtime -Isrc -Igenerated src/main.cpp src/ASTNodes.cpp src/ASTBuilder.cpp src/ASTPrinter.cpp src/LogicalPlanNodes.cpp src/LogicalPlanBuilder.cpp src/LogicalPlanPrinter.cpp src/PhysicalPlanner.cpp src/PhysicalOperator.cpp src/ExecutionBuilder.cpp generated/GQLLexer.cpp generated/GQLParser.cpp generated/GQLBaseVisitor.cpp -lantlr4-runtime -L/usr/local/lib -o gqlparser && ./gqlparser tests/test_match.gql


3. Test a Query

./gqlparser test_query.gql



4.Terminal 1 (Backend API):


cd web/backend
node server.js

5.Terminal 2 (Frontend UI):

cd web/frontend
npm run dev
